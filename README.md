# ChangeVol


RU

Простое Android-приложение для управления громкостью мультимедиа. Отображает 5 больших цветных кнопок с фиксированными уровнями громкости (30%, 40%, 50%, 60%, 75%). Предназначено для использования в Android-магнитолах и автомобильных устройствах для быстрого изменения громкости в одно нажатие. Например если Яндекс музыка предлагает треки разной громкости.

EN

Simple Android app for controlling media volume. Provides 5 large color-coded buttons with fixed volume levels (30%, 40%, 50%, 60%, 75%). Designed for Android head units and in-car devices for quick one-tap volume adjustment. For example, if a streaming service offers tracks of different volumes.
